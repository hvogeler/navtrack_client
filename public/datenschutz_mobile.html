In dieser Datenschutzerklärung möchte ich, Heiko Vogeler, Alteburger Str. 212, 50968 Köln, Deutschland, die Nutzer meiner App Navure darüber informieren, inwiefern personenbezogene Daten erfasst, gespeichert und verarbeitet werden.

Welche personenbezogenen Daten werden erfasst?

Tracks

Sinn von Navure ist es, den Benutzer anhand eines GPS Tracks (=Datei mit GPS Punkten) dabei zu unterstützen, seinem gewünschten Weg in der Natur zu folgen. Dazu wird der Track auf dem IPhone gespeichert. Je nachdem wie ein Track erstellt wurde, kann er personenbezogene Daten beinhalten. Welche Daten das genau sind, hängt davon ab, wie der Track erfasst wurde. Da Navure selbst nur fertige Tracks speichert, nicht aber welche erstellt oder ändert, hat Navure keinen Einfluss auf die inhaltlichen Details der Tracks, die der Benutzner speichert.

Abruf von Tracks von www.navure.hvo.de

Der Benutzer kann in Navure Tracks suchen, der auf meiner Web Seite www.navure.hvo.de gespeichert sind. Wenn er das tut, wird die IP Adresse, die das IPhone im Moment der Suche hat, an meinen Web Server übermittelt und dort gespeichert. Mit dieser IP Adresse werden nur technische Daten wie Datum/Uhrzeit der Abfrage sowie der Suchbegriff gespeichert. Diese Daten dienen nur zur Erkennung von Spam Abfragen und werden nach 6 Monaten gelöscht.

Auf welcher Rechtsgrundlage und wie lange verarbeite ich diese Daten?

Rechtsgrundlage für die Verarbeitung ist mein berechtigtes Interesse nach Art. 6 (1) lit. f DSGVO, den oben beschriebenen Dienst, der Navigation entlang eines vorgegeben Tracks, für Sie zu erbringen. Die Daten werden nach 6 Monaten wieder gelöscht.

Weitergabe der Daten

Die Daten werden auf einem Server meines Web Dienstleisters 1Blu gespeichert.

Sicherheit

Es handelt sich um einen dedizierten Server, auf den nur ich Zugriff habe.
